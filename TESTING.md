Browser Testing Checklist

1. Open index.html in Chrome/Edge/Firefox
2. Verify canvas renders and snake appears
3. Controls: Arrow keys/WASD move snake; Space pauses/resumes
4. Touch: on mobile, use on-screen arrows to move
5. Speed slider: changing Delay adjusts game speed
6. Dynamic difficulty: enable and verify speed increases with score
7. Sound: press Start then toggle Music; eat food to hear eat tone
8. High score: get a score, refresh page, confirm high score persisted
9. Packaging: run `package.bat` to create Aviator.zip

If any test fails, open DevTools Console and report errors.
